## 🎯 Mục Tiêu Khóa Học

- **Nền tảng vững chắc:** Làm chủ cú pháp, kiểu dữ liệu và cấu trúc điều khiển trong Java.
- **Tư duy OOP:** Áp dụng triệt để Lập trình Hướng đối tượng (Đóng gói, Kế thừa, Đa hình, Trừu tượng).
- **Kỹ năng thực chiến:** Xử lý ngoại lệ (Exception Handling), thao tác với tập tin (I/O Streams) và cấu trúc dữ liệu (Collections Framework).
- **Code sạch & Tối ưu:** Rèn luyện thói quen viết code dễ đọc, dễ bảo trì và tối ưu hiệu suất.
